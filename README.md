NutriBot is a web-based nutrition assistant chatbot that helps users explore nutritional information for various food items and beverages. It allows users to search food nutrition, compare items, and get recommendations through an interactive chat interface.

The application is built using a Python backend and a responsive web frontend, providing a simple and user-friendly experience.

---

## Live Demo

https://nutri-chatbot-mxd5.onrender.com

---

## Features

* Interactive chatbot interface
* Search nutritional information for food items
* Compare nutritional values between foods
* Get recommendations such as high-protein or low-fat items
* Real-time chat responses
* Clean and responsive user interface
* Cloud deployment support

## Installation and Setup

Clone the repository

```id="nutri-clone"
git clone https://github.com/yourusername/nutribot.git
```

Navigate to project folder

```id="nutri-cd"
cd Nutri-chatbot
```

Install dependencies

```id="nutri-install"
pip install -r requirements.txt
```

Run the application

```id="nutri-run"
python app.py
```

The application will run on

```id="nutri-url"
http://localhost:5000
```
